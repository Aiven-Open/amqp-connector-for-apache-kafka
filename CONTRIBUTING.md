# Welcome!

Contributions are very welcome on the Aiven framework for connectors on Apache Kafka.

Please see our [contributing documentation](https://aiven-open.github.io/aiven-kafka-connector-framework/contributing.html) on the documentation site.

When contributing please keep this in mind:

- Open an issue to discuss new bigger features.
- Write code consistent with the project style and make sure the tests are passing.
- Stay in touch with us if we have follow up questions or requests for further changes.
